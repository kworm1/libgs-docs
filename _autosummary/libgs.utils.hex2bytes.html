libgs.utils.hex2bytes¶

libgs.utils.hex2bytes(hexstr)[source]¶

Helper function to convert a hex string (AA-BB-CC-…) to a python bytearray

Example:

>>> hex2bytes("AA-BB-00-01-10")
bytearray(b'\xaa\xbb\x00\x01\x10')
