libgs.hardware.RotatorBase¶

class libgs.hardware.RotatorBase[source]¶

Base class for any rotator hardware interface

Any rotator interface must derive from this class and implement the interface methods:

  • get_azel()
  • set_azel()

Additionally you should set the name attribute to something descriptive, and change the configuration attribute parameters as appropriate:

  • STOWED_AZ
  • STOWED_EL
  • BEAMWIDTH
  • MAX_AZ
  • MIN_AZ
  • MAX_EL
  • MIN_EL
  • SLEW_TIMEOUT

Methods

azel_err(az, el) Returns the error in degrees between the current pointing and a specified az/el
azel_to_antenna_angles(pdat[, cont_track_method]) From a table of az/el pointings, compute a new table that minimises the amount of movements while keeping the off-pointing within the antenna beamwidth
get_azel() Get current az, el pointing
in_pos([az, el]) Check if antenna is in position.
set_azel(az, el, block) Set new az, el pointing
stow([block]) Stow antenna
