libgs.database.KVDb.get¶

KVDb.get(limit=None, keys=None, tstamps=None, **kwargs)[source]¶

Get data from the database.

Filter by any column, the where query is constructed as WHERE <column> like <…> AND <column> like <…>

where the column and … come from the kwargs. You can do partial matches using the % wildchar. e.g. keys = ['%NW%', '%NE%'] will match all keys with NW and NE in their names.

Parameters:
  • tstamps – Timestamp filter. See tfilterstr2query()
  • **kwargs – Other column filters
  • limit – limit to a number of results
Returns:

DataFrame containing the stored data
