libgs.visualisation.Markup¶

class libgs.visualisation.Markup(name, text, mtype='PreText', **args)[source]¶

A LivePlot wrapper for the three Bokeh Markup widgets;

  • bokeh.models.widgets.markups.Div
  • bokeh.models.widgets.markups.Paragraph
  • bokeh.models.widgets.markups.PreText

The text (or html) can be changed through the ‘text’ property in the live_props attribute.

Parameters:
  • name – The LivePlot.name for the LivePlot
  • text – The text/html to populate the widget with
  • mtype – The type of Boke Markup widget. One of ‘LivePlot’, ‘PreText’, or ‘Paragraph’
  • **args – Additional kw arguments to pass to the bokeh widget constructor.
